# Scalper OLX

Prosta i skuteczna aplikacja do wyszukiwania i śledzenia ofert na OLX.

## Funkcje

*   **Wyszukiwanie ofert**: Wpisz frazy (np. "iphone 13, ps4 pro") i znajdź najnowsze ogłoszenia.
*   **Filtrowanie i Sortowanie**: Ustaw maksymalną liczbę ofert, wybierz widok (siatka/lista) i sortuj po cenie.
*   **Historia**: Wszystkie znalezione oferty są zapisywane w lokalnej bazie danych, co pozwala śledzić historię cen.
*   **Wizualizacja**: Wykresy i statystyki znalezionych ofert.
*   **Nowoczesny Wygląd**: Przejrzysty interfejs z "przyklejonym" paskiem wyszukiwania.

## Wymagania

*   Python 3.8 lub nowszy
*   Połączenie z internetem

## Instalacja

1.  Pobierz ten folder.
2.  Uruchom plik `install.bat`, aby automatycznie zainstalować wymagane biblioteki.
    *   Alternatywnie otwórz terminal i wpisz: `pip install -r requirements.txt`

## Uruchomienie

1.  Kliknij dwukrotnie plik `run.bat`.
2.  Aplikacja otworzy się w Twojej domyślnej przeglądarce internetowej.

## Użycie

1.  Wpisz szukane frazy w polu na górze (oddzielone przecinkami).
2.  Dostosuj filtry po lewej stronie (opcjonalnie).
3.  Kliknij przycisk "Szukaj".
4.  Przeglądaj wyniki w formie siatki lub listy.
